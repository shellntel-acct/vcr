The Center for Information Security (CIS) is an organization dedicated to improving the global security posture of both public and private sectors. CIS produces consensus-based, best practice secure configuration benchmarks and security automation content, and serves as the key cyber security resource for state, local, territorial and trial governments, including chief security officers, homeland security advisors and fusion centers.

Details

The CIS audit evaluates specific configuration settings against best practices for that operating system. The results are divided up into one of several categories:

PASSED The best pratice setting ("policy value") and the actual matching setting on the target system ("actual value") are either the same or the actual value is more secure than the policy value.
FAILED The actual value does not match the policy value or is less secure than the policy value.
ERROR An error was encountered attempting to evaluate the setting in question. Often times this can be due to insufficient permissions of the account running the check, or another misconfiguration which prohibits the evaluation of the setting.
INFO An informational message that simply conveys information about the environment or a particular check. Information messages do not affect the percentages in the "At-A-Glance" chart on the left.
